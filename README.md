# 台股當沖損益計算機 (Taiwan Stock Day-Trading Profit Calculator)

這是一個專為台灣股市交易員設計的當沖損益計算工具，提供極度精確的計算結果，包含券商手續費折數設定與多平倉位價格試算。

## 功能特點

- **多空雙向支持**：支持「做多 (先買後賣)」與「做空 (先賣後買)」兩種交易模式。
- **靈活配置**：可自定義券商手續費折數（如 2.8 折）。
- **損益兩平計算**：自動計算平攤所有成本後的兩平價格。
- **獲利點試算表**：快速查看不同漲跌幅下的預期淨損益與報酬率 (ROI)。
- **專業主題**：採用深色模式（Dark Theme），符合專業交易工具美學。
- **極速反應**：純原生 JavaScript 開發，無任何外部庫依賴，加載迅速。

## 如何使用

1. 輸入**成交價**。
2. 輸入交易**張數**。
3. 設定您的**券商折數**。
4. 切換**做多/做空**模式。
5. 即時在下方表格查看對應的平倉獲利結果。

## 技術實現

- **HTML5**: 語義化結構，對 SEO 友好。
- **Vanilla CSS**: 自定義變量、Flexbox/Grid 佈局，現代感 UI 設計。
- **Vanilla JavaScript**: 高性能計算邏輯，無冗餘框架負擔。

## 免責聲明

本工具僅供參考，計算結果可能因小數取捨或特殊手續費規定而有微小差異。投資理財請審慎評估風險，交易前請以券商系統顯示之數據為準。
